# Human Resource Management System (HRMS)

## Project Description

The Human Resource Management System (HRMS) is a Core Java console-based application developed to manage employee records, attendance, and leave details efficiently. The system helps HR administrators perform employee-related operations easily through a menu-driven interface.

## Features

* Employee Management
* Attendance Tracking
* Leave Management
* Employee Search
* Report Generation
* User Authentication
* Error Handling

## Technologies Used

* Core Java
* OOP Concepts
* ArrayList Collection
* Scanner Class
